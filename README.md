# RFID-Reader

Portuguese:
Leitor RFID 

Este leitor permite ler qualquer leitor Mifare 13,56MHZ. Simulando a digitação do seu UID em Decimal.

Exigencias:
Arduino Leonardo.
- Arduino Leonardo.
- RFID-RC522

PINOUT :
ARDUINO    <--->   RC522
3.5v                 3.5v
GND                  GND
pin10                RST
PIN4(GND)             -

- RFID-RC522
 

PINOUT :

3.5v                 3.5v
